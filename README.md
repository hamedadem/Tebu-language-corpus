# Tebu Language Open Corpus

This repository contains an open and curated corpus of the Tebu (Toubou) language,
written using a standardized Latin orthography.

## Language Information
- Language name: Tebu (Toubou)
- Writing system: Latin
- Regions: Libya, Chad, Niger
- Language family: Nilo-Saharan (Saharan branch)

## Contents
- Monolingual Tebu texts
- Parallel translations (Arabic, English, French)
- Dictionaries and glossaries
- Metadata and language documentation

## Purpose
This corpus supports:
- Language preservation
- Academic research
- Machine translation and AI tools
- Digital inclusion of the Tebu language
